1. repository 생성

Github.com 과 local(자신의 컴퓨터)에 repository를 생성

github에 repository생성시 repository name은 'userName.github.io'로 생성

github desktop을 이용해 github의 repository를 local(자신의 컴퓨터 내부)로 clone

2. Jekyll Themes 적용

http://jekyllthemes.org

원하는 theme를 선택하여 download하여 local repository에 저장한다

3. Ruby + devkit 설치

https://pages.github.com/versions

GitHub Pages가 사용하는 ruby version을 확인하고 해당 버젼을 설치한다

https://rubyinstaller.org/downloads에서 download하여 설치

> ruby -v
ruby 2.5.5p157 (2019-03-15 revision 67260) [x64-mingw32]
						
> gem -v
2.7.6.2

ruby + devkit 을 설치하고 window > cmd 을 열어 ruby와 gem이 설치 되었는지 확인한다

> cd [local repository]
			
[local repository] > gem install jekyll
25 gems installed
			
[local repository] > jekyll serve
Auto-regeneration: enabled for '[local repository]'
Server address: http://127.0.0.1:4000/
Server running... press ctrl-c to stop.

gem을 통해 jekyll을 local repository에 설치하고 jekyll server를 구동

address를 통해 로컬상에서 브라우저로 접속하여 사이트가 어떻게 생성될지 미리 살펴볼 수 있다
